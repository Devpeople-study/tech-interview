# RESTful API 란?

REST는 소프트웨어 프로그램 아키텍쳐의 한 형식입니다.                                                                    

웹의 기존 기술과 HTTP 프로토콜을 그대로 활용하므로 웹의 장점을 최대로 활용할 수 있는 아키텍쳐 스타일로,

RESTful API란 HTTP 통신을 REST 설계 규칙에 맞게 설계하는 API를 뜻합니다.

</br>
